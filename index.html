🖥 ① 机身屏幕 19项
▶3内屏掉漆/划伤重点
📝 质检手法若检查到内屏掉漆/划伤，使用菲林卡尺测量出直径并点清数量，在APP界面勾选对应选项。
▶4屏幕裂痕检查重点
📝 质检手法若检查到裂痕需使用菲林卡尺测量出直径并点清数量。
▶5屏幕支架破损检查重点
📝 质检手法若检查到支架破损需使用菲林卡尺测量出长度。
判定：支架只磕碰未破损算外壳磕碰，有破损的算支架破损。折叠屏机型的屏幕保护胶条也算屏幕支架。
▶6音频网罩检查
📝 质检手法检查音频网罩是否变形/破损或缺失，在APP界面勾选对应选项。
▶7屏幕凸起/褶皱检查注意豁免
📝 质检手法若检查到凸点或褶皱，使用菲林卡尺测量出直径并点清数量。
豁免范围：安卓机型距离屏幕边缘10mm范围内的一圈、屏下指纹模块边缘5mm范围内的一圈为豁免范围。小窗屏和折叠机主屏不看褶皱。
▶8屏幕盖板打磨凹陷检查
📝 质检手法使用530瓶子垂直贴紧屏幕检查是否可看到缝隙或对向透光 → 若能看到需将0.3mm塞规放入缝隙判定。仅直面屏机型检查；曲面屏机型只检查波浪形纹路。OPPO Find X8系列不执行。
▶9折叠机折痕/凹印/凸起检查重点
📝 质检手法若检查到折叠机有发白折痕、凹印或凸起，根据检查情况勾选。
判定：发白折痕需区分轻微和明显；凹印需直径＞3mm才判定；主屏凸起直径需＞1mm，折叠处凸起不算。
▶10折叠机保护膜检查
📝 质检手法检查折叠机屏幕是否存在保护膜 → 判断是否为原厂 → 检查贴膜情况（保护膜上的划痕/气泡/进灰按屏幕标准勾选）。
▶11折叠机支架缝隙检查
📝 质检手法折叠机展开后，使用0.3mm塞规检测左右两边4个位置。塞规需与屏幕平行，不可斜插或过力。若触发支架破损/缺失>2mm则无需检测此项。
▶12折叠屏鼓包/蝴蝶扣检查新版B3新增
📝 质检手法折叠屏折叠过程查看是否存在局部鼓胀、外拱现象。完全展开查看蝴蝶扣是否存在胶条凹陷、变形、翘起。
▶13屏幕白底拍照（折叠机）
📝 质检手法打开拍照手机【质检小工具】→ 点击【质检拍照】→ 扫描二维码 → 1倍焦距下点击拍照 → 确定上传。折叠机需展开主屏+副屏各拍一张白底照。
▶14屏幕显示检查（黑白灰三色）重点
判定标准手机距眼睛15-20cm，APP默认亮度下切换黑、白、灰纯色界面检查。每个界面至少停留4s。亮度过低需调到70%以上。
检查项进灰、气泡、坏点、色斑、漏液、黑色背景有色块、白底局部发暗、条纹、亮点/亮斑
易漏原因切换界面太快（<4s）、亮度不够70%看不出暗病、只检查部分颜色界面
"黑白灰各4秒，亮度七成不能少"
📝 质检手法切换黑→白→灰纯色界面，每界面至少停留4秒。默认亮度下检查，亮度过低时调整到70%以上。
▶15OLED屏隐藏暗线检查易漏
📝 质检手法OLED屏检查条纹时，双手同时按压屏幕上/下方中间位置，看是否会出现隐藏暗线。屏碎及漏液的不执行此操作。
▶16屏幕泛红/泛黄检查
📝 质检手法眼睛距离手机30cm，在肉眼可见泛红/泛黄情况下，使用手机拍照来区分不可拍/可拍。
▶17透图/状态栏分层检查
📝 质检手法检查是否有透图及状态栏分层。状态栏外透图需使用菲林卡尺测量直径。肉眼可观察到时需iPhone 12/13 Pro拍照区分可拍/不可拍。
▶18屏幕黑边/红线检查重点
📝 质检手法使用手电辅助检查屏幕四边黑边是否对齐、黑边宽度是否一致。四边各有一条红线的机器，检查红线是否缺失（直面屏正视查看，瀑布屏侧边查看左右红线）。
▶19屏幕材质/偏光片/支架检查重点
📝 质检手法息屏下借助手电筒检查屏幕材质。紫光灯距离屏幕5-10cm照射检查底色。偏光片端正置于显示区域上方3cm内，正视停顿观察孔位颜色。
判定标准苹果6/7/8号孔深蓝色；安卓12号孔黑色或深蓝色。折叠机只检查副屏。
▶18重点易漏安卓-屏幕拆修情况
▶1外屏-有胶水溢出/撬痕
▶2外屏-感应孔样式与示例不符
▶3外屏-APP黑边红线有缺失
▶4外屏-黑边宽度不匹配（app未配置红线缺失才有此项）
▶5外屏-紫光灯照射边缘内屏异常（华为系列）
▶6工具-屏幕偏光检测异常（安卓）
▶7外屏-触摸键灯局部不亮
▶8屏幕-屏幕材质不符
▶9昆仑玻璃检查（华为机型）

🔩 ② 机身后壳 16项
▶1整体外观目视检查
📝 质检手法视线正对后壳，转动手机在不同角度下检查整个手机背面 → 翻转检查边框。
检查范围：边框包括卡托、尾插、按键、耳机孔、屏幕支架。折叠屏需展开检查。
原则：目视发现瑕疵或不确认异常，需用手触摸确认。
▶2裂痕检查重点
判定标准金属外壳裂纹≤0.3mm算磕碰，＞0.3mm算碎裂；塑料后壳裂纹用指甲判定是否有触感，有触感算碎裂，无触感算磕碰。
注意事项质检项长度相同时勾选较轻的项，如裂痕长度为2mm时勾选【≤2mm】，而不是【2mm-10mm】。
"裂痕必量，金属0.3塑料看触感"
📝 质检手法检查到裂痕需使用菲林卡尺测量裂痕两端之间最长的距离，然后勾选对应选项并提交。
▶3刻字检查
📝 质检手法若后壳检查到有刻字，区分非官方刻字/官方刻字。
判定：【刻字小于50%】或【刻字超过50%】是指非官方刻字总面积占后壳总面积比例，不以字体分布区域判定。
▶4褶皱检查
📝 质检手法若检查到褶皱需使用菲林卡尺测量出长度并勾选对应选项。
▶5机身弯曲变形检查重点
判定标准外壳卡托、尾插、按键、耳机孔周围等有凸起的变形都算机身弯曲。
注意事项后盖摄像头模组较大无法紧贴桌边时，以屏幕面紧贴桌边检测。四角为R角的手机需在距翘起顶部5mm处测量。
📝 质检手法手机边框对着眼睛方向旋转查看是否弯曲变形 → 较长直边紧贴桌边 → 菲林卡尺测量桌边到翘起顶部的距离。
▶6脱胶/缝隙检查重点易漏
判定标准塞规不可斜插或使用边角插。检查区域包含所有外壳与其他部件连接处。
"塞规平行放，轻松能入就算脱"
📝 质检手法用手捏住0.3mm塞规顶部位置平行放入缝隙，若能轻松放入则判定【脱胶缝隙≥0.3mm】，若感受到阻力且塞规弯曲则判定【无】。
▶7摄像头镜头进灰检查易漏
📝 质检手法保持眼睛距手机30cm，顶部向上倾斜45°，左右偏移手机检查前后摄镜头内是否有进灰。若检查到进灰，使用菲林卡尺测量灰尘直径并点清数量。
▶8后壳进灰检查
📝 质检手法若检查到手机后壳内有灰尘，在APP界面勾选【后壳进灰】。
豁免：灰尘直径≤0.5mm且数量≤5粒不算后壳进灰。
▶9外壳缺失/Logo掉落
📝 质检手法若手机外壳完全缺失或嵌入式logo掉落，在APP界面勾选【外壳缺失/logo掉落】。
▶10部件缺失检查
📝 质检手法检查尾插金属保护圈、摄像头保护罩、顶部红外装置镜片、按键（除home指纹键）是否缺失/断裂/脱落，若有勾选【其他异常】。
▶11后盖与镜头保护罩连接处胶水溢出重点高漏检
标准连接处无胶水溢出痕迹，缝隙均匀无白点。
怎么看镜头朝上，45°侧光，用指甲沿连接缝滑动一圈。
"转一圈，摸一圈，缝隙白点不能过"
▶12后壳漆面受损重点高漏检
标准漆面完整，无掉漆/划痕/凹陷/起皮。
怎么看正面强光确认无明显瑕疵 → 侧光30°检查漆面平整度。
"先平面后边角，侧光看漆面"
▶13感应孔样式检查重点高漏检
标准感应孔形状/位置/数量与当批次示例一致。
怎么看拿示例机/照片并排对比，确认开孔位置和形状。
"不对示例只看实物，十个有九个会出错"
▶14按键处溢胶检查重点高漏检
标准音量键/电源键边缘无缝、无胶水残留。
怎么看按压按键 → 观察按键缝隙是否有胶挤出；手指沿按键边缘轻摸。
"按一下，摸一圈，黏手就不能过"
▶15后壳更换检查
📝 质检手法检查手机后壳是否被更换过。三星折叠屏W23-25借助紫光灯检查。
▶16后摄/闪光灯/麦克风网检查重点
📝 质检手法检查后摄印记或镜片缺失 → 闪光灯是否偏移或镜片缺失 → 后摄保护圈是否缺失或镜圈缝隙变大 → 麦克风网是否缺失。
▶17安卓-机身维修情况
▶1后壳-与中框有溢胶/撬痕
▶2后壳-样式不匹配
▶3后壳-壳内漆面受损
▶4后壳-后盖与镜头保护罩连接处有胶水溢出
▶5后壳-后盖与摄像头缝隙大于 0.5mm
▶6后盖-闪光灯有偏移大于0.3mm
▶7后壳-扬声器网/麦克风网缺失
▶8后壳-贴标盖章【拆机内部】
▶9后壳-导电棉/缓冲棉/石墨贴纸缺 失【拆机内部】
▶10后壳-中框封胶异常【拆机内部】

🛠 ③ 苹果拆修 15项
▶2屏幕/后壳连接处溢胶/撬痕检查重点高漏检
检查范围屏幕/后壳与其它部件连接处、支架与中框连接处（含折叠屏副屏）。
"屏幕朝下看脱胶，指甲刮缝查溢胶"
📝 质检手法目视检查屏幕四边是否脱胶（屏幕朝下检查）→ 屏幕/后壳连接处是否有溢胶/撬痕 → 目视不确认时用指甲刮动确认。
▶3感应孔样式与示例不符检查重点
📝 质检手法眼睛正视手机，目视检查感应孔是否和原厂不符。苹果机型使用手电筒辅助检查。折叠屏大副屏需检查，小副屏不需检查。
▶4原彩显示检查（苹果机型）
📝 质检手法下拉状态栏长按亮度条 → 查看是否有原彩图标。iPhone 8及以后机型有原彩功能。缺失或没有 →【屏幕-无原彩显示】。
▶5爱思助手验机检查易漏
📝 质检手法APP界面靓机助手回显异常 → 插线连接电脑打开爱思助手查看验机报告是否有拆修记录。
▶6未知部件提示检查
📝 质检手法状态栏或关于本机出现未知部件等提示 → 勾选对应部件异常。
华为Mate 60/Pura 70/Mate 70系列：连接充电器 → 更新应用 → 连接充电器2-3次检查是否弹窗。
▶7电池健康度检查
📝 质检手法进入设置 → 电池 → 电池健康 → 截图 → 关于本机 → 截图 → 上传。B2C卖场手机电池健康度<83%需更换电池。
▶8昆仑玻璃检查（华为机型）重点
📝 质检手法配置【更换非昆仑玻璃】项的华为机型，在息屏状态下使用手电（白光）照射屏幕盖板，检查颜色是否为蓝色系，若不是判定为更换非昆仑屏。
▶9卡槽/扬声器/尾插/按键开孔弧度检查重点
📝 质检手法用卡针取出卡托 → 手电辅助检查卡槽颜色、开孔弧度、小型号及是否更换卡座 → 检查所有扬声器和麦克风网是否缺失、开孔弧度是否正常 → 检查尾插及螺丝是否缺失/更换/损坏 → 检查所有按键是否缺失及开孔弧度是否正常。
华为机型：需检查中框切面(指纹键区域)交接处是否有明显台阶感。
▶10iPhone17Pro/17ProMax外壳翻新检查新版B3新增
📝 后壳检查紫光灯贴紧后玻璃，查看颜色是否为紫色/蓝色（星宇橙色需微微抬起紫光灯露出"月牙"光区查看，深蓝色需用手遮挡周围光线）。
📝 充电口检查查看是否有一圈高光处理的金属圈嵌入充电口内。
📝 闪光灯/麦克风倒角斜看是否存在斜切面/高光倒角。后置麦克风斜看是否存在漏斗形状。
📝 刀纹检查使用放大镜查看中框与后玻璃接合处的中框边缘有无刀纹，无刀纹为原装。
▶11手工拆机操作重点
📝 操作流程拧下底部螺丝 → 检查螺丝是否缺失/凸起/滑丝/不符 → 15-20cm目视检查缝隙 → 不确认时用刀片沿缝隙刮查溢胶 → 刀片垂直插入长边（LCD屏不能滴酒精）→ 刀片翘起拉缝 → 拨片插入沿长边划动 → 刀片从顶部插入翘起 → 从卡槽另一边小心打开手机（张开角度≤90°）。
⚠ 关键注意拆机时保持屏幕常亮；插入拨片不能太深以防滑断排线；下刀避开按键和卡托位置。
▶12内部拆修痕迹检查重点
📝 检查清单1. 内部是否有维修标识贴纸或盖章
2. 贴纸/海绵/零部件是否缺失、破损、更换或维修
3. 排线是否翘起或飞线，是否有焊油/焊锡残留
4. 内部固定螺丝是否缺失、型号不符或滑牙
5. 防水标是否变红，内部有浸液发霉/生锈
屏幕IC检测紫光灯距离屏幕IC 5cm范围内照射，观察是否呈现荧光青（适用12-17系列）。
▶13更换电池操作（B2C卖场健康度<83%）
📝 操作流程取下电池排线螺丝和屏蔽罩 → 镊子拨开电池排线 → 电池边缘滴酒精 → 镊子轻轻翘起电池（避开排线）→ 清理残胶 → 领新电池贴电池胶 → 扣排线接口 → 按压电池四边角 → 装回屏蔽罩螺丝。
注意：翘电池力度适中，禁止暴力撬。一定要避开底部排线。
▶14贴内部防拆标
📝 操作质检码带星号的需贴内部防拆标 → 贴在电池接口和屏幕排线接口位置并覆盖螺丝。若内部已有友商标签直接覆盖。
▶15激光雷达检查
📝 质检手法使用iPhone 13 Pro录像模式+闪光灯常亮+微距模式，距离5cm左右查看颜色是否正常（iPhone17Pro/17ProMax不检）。

⚙ ④ 功能检查 16项
▶1充电功能检查重点
📝 质检手法检查尾插接口是否破损 → 若未破损连接充电器（最多同时操作4台）→ 捏住充电线左右晃动4次、上下晃动2次（Type-C和Lighting两面都查）。
快充检查：华为Mate 40 Pro/Mate X5/Mate 60系列/Pura 70系列必须使用原装充电器检测超级快充。
判定：出现受潮/潮湿/无法充电提示 → 充电【异常】。尾插破损 → 直接判定充电异常。
▶2充电器/线检查
📝 质检手法检查充电器/线是否存在破损、接口内部是否存在断裂/变形。充电线有压痕需查看对折处是否有破损至裸露线材。破损/断裂/变形 → 拦截。
▶3爬虫信息核对重点
📝 质检手法核对首页爬虫信息，检查本机实际情况和爬虫信息是否相符，勾选本机基本情况和系统相关选项。
注意：ROOT、BL锁和华为监管机以APP回显结果为准。爬虫信息标红的需重点核实。全红时需重新获取。
▶4SIM卡槽/卡托检查易漏
📝 质检手法取卡针插入卡槽孔 → 卡托弹出 → 检查卡槽孔内是否有异物、卡托是否突出或表面断裂变形 → 取出卡托检查是否更换（查看IMEI和颜色）→ 装入测试SIM卡（优先电信卡，双卡机一个必须是电信）。
▶5振动 & 按键功能检查
📝 质检手法点击振动按钮检查振动反馈 → 连续按压3次音量+键 → 连续按压3次音量-键 → 拨动3次静音键 → 按压1次电源键。若有其他按键需连续按压/滑动3次确定功能正常。
▶6闪光灯检查
📝 质检手法侧翻手机查看闪光灯是否开启且颜色正常。无法打开需返回桌面下拉打开闪光灯核实。部分机型电量<15%无法开启，需补电后再检。
▶7相机功能检查（前后摄）重点
📝 对焦检查后摄对准近处参照物，在默认倍焦和广角下查看对焦 → 对准远处参照物，切换广角/1X/长焦/人像模式检查。
📝 坏点和有斑白色卡片遮挡后摄/前摄，白底下观察 → 若有斑，用红色卡片遮挡红底下复核。先确认配置是"白底有斑"还是"红底有斑"。
易漏原因切换倍焦太快没等画面稳定、只查后摄不查前摄、白底看到斑没做红底复核
▶8音频功能检查（听筒/扬声器/麦克风）重点
📝 质检手法切换听筒播放/扬声器播放检查异常 → 录音对着麦克风喊"123" → 播放录音检查异常和送话弱情况。异常时使用本机录音软件复核。
▶9ID检查 & 账户解锁重点
📝 质检手法跳转设置页面 → 查看是否存在ID并截图 → 生物识别和密码 → 录入指纹并验证 → 录入面容并验证 → 设置密码000000。
▶10通话/SIM卡信号检查重点
📝 质检手法插入SIM卡 → 拨打运营商客服电话（电信10000/移动10086/联通10010）→ 检查是否无法通话。
▶11触摸功能检查
📝 质检手法用力按压5个按钮 → 滑动屏幕将所有灰色触点变绿。安卓虚拟按键也属于触摸功能。折叠屏需展开检查主屏+合拢检查副屏。
▶12WIFI/蓝牙/NFC/无线充电检查
📝 质检手法根据质检过程判定WIFI/蓝牙是否正常 → 启用NFC → 将手机放NFC感应区域 → 观察NFC和无线充电功能。
▶13指南针/重力感应/距离感应/光线感应
📝 质检手法转动手机检查指南针和重力感应 → 用手遮挡手机顶部检查距离感应和光线感应。
▶14折叠机转轴/悬停/屏幕切换检查新版B3新增
📝 质检手法开合折叠机至90°左右检查悬停功能 → 展开2秒看主屏是否亮屏 → 闭合2秒看副屏是否亮屏（OPPO Find N/N2不检测悬停）。转轴无法完全合拢/打开、异响、松动、卡顿→【转轴异常】。
▶15手写笔检查（部分机型）
📝 质检手法取出手写笔检查外观 → 测试功能按键 → 在屏幕上滑动检查功能 → 插回检查是否可完全插入。
▶16游戏手机散热风扇/呼吸灯检查
📝 质检手法检查散热风扇是否转、有异响 → 查看呼吸灯是否有节奏亮起（仅部分机型检查）。

📱 ⑤ 扫码质检 & 拍照入库 14项 · 必备流程
▶1取机 & 清理工作台
📝 操作在货架对应区域按从左往右顺序拿取机器。质检区域最多出现2筐机器，调拨时最多12台每次。
▶2连接WIFI & 进入质检APP
📝 操作解锁手机 → 点击【请选择WIFI】→ 连接 → 点击【ZZ_Android】（部分站点是 ZZ_Apple）→ 点击【允许】→【始终允许】→【手动检测】。
注意：若跳转出视频播放界面，需对手机进行老化测试。
▶3扫描货位二维码 & 上架扫码
📝 操作取手机扫描视频播放架二维码 → 手机界面显示对应货位信息 → 将手机放到对应货位并连接数据线。
指示灯规则等待连接：亮黄灯 · 连接成功：亮蓝灯 · 完成老化：亮绿灯 · 异常：亮红灯交组长处理
电量规则低于30%：自动充电至30%以上 → 播视频10分钟 · 30%-80%：充电3分钟 → 播视频10分钟 · 高于80%：充电3分钟 → 放电到80%以下 → 播视频10分钟
▶4登录质检系统
📝 操作扫描员工ID二维码进入质检页面。无法扫码时，手动输入员工ID，点击【确认】。
▶5系统版本检查（苹果15/16系列）易漏
📝 操作苹果15、16系列进入质检界面若出现版本低于iOS26的提示，需通过爱思助手升级到iOS26及以上。
注意：上传关于本机图片时需对更新后的内容进行上传；系统更新后若报告中版本未更新，需人工修改。
▶6清洁后壳（预备外壳检查）
📝 操作按压酒精喷头打湿无尘布 → 用无尘布从下往上来回擦拭后壳一遍（包含边框及后摄镜头），保证外壳洁净。顽固污渍使用530清洗剂。
▶7提交质检报告重点
📝 操作系统对质检报告进行判定 → 如需要拍照则弹出拍照二维码 → 连续点两下【双击提交】提交报告。
▶8瑕疵拍照重点
📝 操作点击【瑕疵&主板拍照】→ 扫描拍照二维码 → 在质检手机上找到对应瑕疵项 → 拍照手机切换到2倍焦距 → 白色背景下对准瑕疵部位拍照 → 手指滑动框选最严重的瑕疵 → 确定。
拍照要求拍照前擦拭瑕疵位置；选取同类瑕疵中最严重的；横平竖直调整角度；不允许出现二维码、人脸、人手
▶9拆机留底图拍照（苹果12-16系列）
📝 操作返回小工具界面 → 点击【留档图】→ 点击【主板图】→ 扫描质检码 → 按标准示例图2倍焦拍摄 → 提交。
▶10手工装机易漏
📝 操作屏幕对齐摄像头/按键等孔位轻轻装入机身 → 按压四边使紧密贴合 → 用螺丝刀将底部螺丝装回。
装机后检查：内部是否有异物或螺丝掉落；屏幕显示/触摸/home键功能完好；装机平整度，四边完全贴合无缝隙且无黑胶溢出。
▶11贴外部防拆标
📝 贴标规则· 普通机：底部+顶部各1张 · 升降摄像头机：2张统一贴机身 · 后置大面积摄像头机：顶部无法贴时贴右侧边 · 上下折叠机：共6张 · 翻盖机：后盖上下各1张+上方屏幕1张 · 外折叠大屏/三折叠机：共6张 · 安卓可拆后盖类：贴主板螺丝位
⚠ 注意事项贴标前清洁位置。不能遮挡屏幕显示区、听筒网、扬声器孔、卡托、卡针孔、感应器孔、送话孔。必须包含一张带二维码的贴在底部或重要位置。
▶12手机还原操作
📝 苹果设置 → 通用 → 传输或还原iPhone → 抹掉所有内容与设置。
📝 安卓设置 → 系统与更新 → 重置手机 → 恢复出厂设置。
⚠ 华为荣耀重启后激活到账户界面检查激活锁提示。荣耀50/50Pro需关机后重新开机激活到账户界面，重复3次。
▶13打印质检码 & 匹配手机盒
📝 操作质检结束后打印质检码贴到对应手机上 → 根据IMEI在货架上找到对应手机盒 → 检查充电器和机器是否匹配、品牌是否一致。
▶14贴充电器防拆标 & 装盒入库
📝 操作充电器防拆标损坏或缺失 → 取防拆标分别贴在充电头和充电线上 → 将充电头和充电线装回手机盒 → 将手机装入手机盒 → 周转入库。
